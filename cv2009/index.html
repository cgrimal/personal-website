Le site de l'AAECPPT

J'ai réalisé ce site à la demande du président de l'association afin de mettre en ligne un annuaire de tous les anciens élèves du CPP de Toulouse, et de toutes les entreprises employant de anciens élèves. Ceci dans le but de regrouper le maximum d'élèves et de leur permettre de trouver des contacts, ainsi que de communiquer sur les événements auxquels participent l'association.

J'ai donc réalisé ce site en utilisant principalement xHTML et CSS pour la présentation, et PHP et SQL pour la gestion de la base de données. J'ai également mis en ligne une partie administration pour ce site, permettant entre autre, de gérer la base de données, les articles publiés dans la section Actualités, ainsi que d'envoyer des mails à l'ensemble des contacts.
En accord avec les "Lois Informatique et Libertés", les contacts peuvent se connecter sur le site puis modifier les informations les concernant.

Le site est en ligne à http://www.aaecppt.fr et a déjà permis de regrouper plus de 300 élèves !

Projet de gestion d'objets dupliqués

Dans le cadre des cours à l'ENSEEIHT sur les systèmes concurrents et communiquants, j'ai réalisé un projet en binôme sur la gestion d'objets dupliqués s'appuyant sur Java RMI pour la communication entre les différents objets Java répartis.

Des applications peuvent donc choisir de partager un objet en l'enregistrant auprès d'un serveur, et une autre application peut ensuite décider de récupérer un objet partagé. Afin d'assurer la cohérence d'un tel système, nous avons utilisé le principe des verrous, en lecture ou en écriture, un objet ne pouvant avoir qu'un seul écrivain ou plusieurs lecteurs au même moment.

Vous pouvez télécharger le sujet du projet (Sujet POD) ainsi que le rapport final que nous avons rendu (Rapport POD).
